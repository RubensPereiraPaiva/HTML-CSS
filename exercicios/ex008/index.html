Principais Formatações

Negrito

Nesta frase uso um termo em negrito usando a tag B (Não Semantico)

Nesta frase, tenho um termo em destaque usando o strong (Semântica)

Itálico

Nesta frase, temos um termo em italico usando a tag I (Não Semantico)

Nesta frase temos um termo em ênfase usando a tag EM (Semântica)

Texto Marcado

Podemos também criar um texto marcado usando a tag MARK

Texto Grande e Texto Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo, usando a Tag BIG para grande (Descontinuada) e SMALL para pequeno

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag DEL

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag INS (Semântica) enquanto a tag U é sublinhado porém não semântico

Texto Sobrescrito

Para inserir coisas do tipo x20+3 usando a tag SUP

Texto Subescrito

Para inserir coisas do tipo H2O usando a tag SUB
